A1. Kelebihan User Interface (UI)

Berdasarkan review tampilan file manager desktop OneDrive Nurhasanah:

  • Desain Bersih & Modern: Menggunakan warna netral (putih & abu-abu muda) yang memberikan kesan rapi dan nyaman dipandang.
  • Struktur Navigasi Jelas: Bagian atas terdapat breadcrumb navigation (OneDrive > Nurhasanah - Personal > Desktop).
  • Tata Letak Terorganisir: File disusun dalam bentuk tabel yang rinci (Nama File, Status, Tanggal Modifikasi, Jenis File, Ukuran File).
  • Ikon Mudah Dikenali: Setiap file memiliki ikon visual berbeda (PDF, Word, Shortcut, Gambar).
  • Fitur Pencarian Mudah Diakses: Kolom pencarian berada di pojok kanan atas secara jelas.

A2. Kekurangan User Interface (UI)

  • Informasi terlalu padat dan terkesan ramai bagi pengguna baru.
  • Ukuran tulisan data rincian relatif kecil untuk keterbatasan penglihatan.
  • Notifikasi pop-up sistem (Snipping Tool) di sudut bawah menutupi sebagian area kerja.

B1. Kelebihan User Experience (UX)

  • Mudah dipelajari langsung tanpa tutorial karena alur navigasi familiar.
  • Akses file efisien dengan bantuan fitur sorting dan sidebar menu cepat di kiri.
  • Integrasi OneDrive yang transparan ditunjukkan melalui indikator status sinkronisasi centang hijau.

B2. Kekurangan User Experience (UX)

  • Terlalu banyak file shortcut menumpuk di desktop memperlambat pencarian optimal.
  • Kurang ramah bagi pengguna lanjut usia yang memerlukan teks ukuran besar.
  • Notifikasi otomatis berisiko mengalihkan fokus tugas utama.

C. Kesimpulan & Skor Akhir

Secara keseluruhan, tampilan desktop memiliki UI yang modern, terstruktur, serta UX yang efisien. Pembenahan pada kepadatan informasi akan menyempurnakan kenyamanan kerja.

Skor UI: 3.5/10

Skor UX: 8.7/10

Skor Keseluruhan: 8.6/10
